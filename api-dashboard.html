API Documentation
ඔබේ app ට AI chat integrate කරන method.
POST /api/chat/v1

Header: x-api-key: YOUR_API_KEY   |   Content-Type: application/json

JavaScript / Node.js
const res = await fetch("/api/chat/v1", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key":    "MALIYA-MD-YOUR_KEY_HERE"
  },
  body: JSON.stringify({
    message:   "Hello! How are you?",
    sessionId: "user-123"   // optional, keeps conversation memory
  })
});

const data = await res.json();
console.log(data.reply);   // AI reply
console.log(data.model);   // which AI model answered
Python
import requests

res = requests.post(
    "/api/chat/v1",
    headers={
        "Content-Type": "application/json",
        "x-api-key":    "MALIYA-MD-YOUR_KEY_HERE"
    },
    json={
        "message":   "Hello! How are you?",
        "sessionId": "user-123"
    }
)

data = res.json()
print(data["reply"])   # AI reply
print(data["model"])   # model used
cURL
curl -X POST /api/chat/v1 \
  -H "Content-Type: application/json" \
  -H "x-api-key: MALIYA-MD-YOUR_KEY_HERE" \
  -d '{
    "message":   "Hello!",
    "sessionId": "user-123"
  }'
Success Response
{
  "ok":    true,
  "reply": "Hello! I'm doing great, thank you for asking! 😊",
  "model": "pollinations",
  "lang":  "en"
}
Error Responses
{ "ok": false, "error": "x-api-key header required." }      // 401
{ "ok": false, "error": "Invalid or inactive API key." }    // 403
{ "ok": false, "error": "Rate limit — wait 2 s..." }        // 429
{ "ok": false, "error": "message field required." }         // 400
Rate Limits
• Requests per session: 2 second cooldown
• Max keys per account: 5
• Message max length: 1000 characters
• sessionId: optional — use same value to keep conversation context
