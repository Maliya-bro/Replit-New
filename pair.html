GUIDE

How to Use

Follow these simple steps to connect your WhatsApp bot in under a minute.

 METHOD 1 — PAIR CODE (Recommended)
1

Go to Pair Code page

Click Pair Code in the navigation menu above to open the pairing page.

2

Enter your number

Type your full WhatsApp phone number with country code — example: 94756XXXXXX for Sri Lanka.

No + sign needed
3

Click Generate Code

Wait a few seconds. A red 8-digit code like ABCD-EFGH will appear on screen.

4

Open WhatsApp → Linked Devices

In WhatsApp, tap the 3-dot menu → Linked Devices → Link a Device → Link with phone number.

5

Enter the code

Type the 8-digit code shown on this site into WhatsApp. Your bot session will be saved automatically.

Code valid for ~60 seconds
6

Done! Copy your Session ID

After linking, your Session ID appears and is saved to the database. Copy it to your bot's config.

Session saved to MongoDB
 METHOD 2 — QR CODE
1

Go to QR Code page

Click QR Code in the navigation to open the QR page.

2

Click Generate QR Code

A QR code will appear on screen within a few seconds.

QR expires in ~20 seconds
3

Scan with WhatsApp

In WhatsApp → Linked Devices → Link a Device — point your camera at the QR code on screen.

4

Session saved automatically

Once scanned, your Session ID is saved to the database. Copy it and use it in your bot config.
